# 喜（悲）报生成器

## 重大更新

字体渲染引擎现已迁移至 `skia-python`，带来了渲染 Unicode 级别的字形和 emoji 支持（人话：字体不支持一个字符时会自动回调至有这个字符的系统字体）

## 安装

```
nb plugin install nonebot-plugin-xibao
```
```
pip install nonebot-plugin-xibao
```

## 使用

使用时记得加上前缀哦！
| 命令 | 功能 |
| --- | --- |
| `喜报` | 生成一张喜报 |
| `悲报` | 生成一张悲报 |
