# 喜（悲）报生成器

## 安装

```bash
nb plugin install nonebot-plugin-xibao
```

或者

```bash
pip install nonebot-plugin-xibao
```

之后在 `pyproject.toml` 里自行添加插件

## 使用

使用时记得加上前缀哦！

| 命令 | 功能 |
| --- | --- |
| `喜报` | 生成一张喜报 |
| `悲报` | 生成一张悲报 |
